# Rejection Rules

Reject a strategy if any of the following are true:

- It only works in one narrow historical period.
- It loses edge after realistic spread/slippage assumptions.
- It has too few trades to support confidence.
- Nearby parameter values collapse performance.
- It depends on ambiguous or discretionary rules.
- It shows signs of lookahead bias or data leakage.
- It requires unrealistic fills during volatile periods.
- It has unacceptable drawdown relative to expectancy.
- Its edge disappears out-of-sample.
- It cannot be explained in simple structural market terms.
