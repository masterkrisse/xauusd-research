# Live Realism Checklist

Review before approving any strategy:

1. Spread spikes during target entry periods
2. Slippage sensitivity
3. Broker/session time differences
4. Fast-bar execution realism
5. Delayed fill sensitivity
6. Rollover or low-liquidity effects
7. Symbol-specific contract or execution constraints

If live-trading realism materially changes the result, downgrade confidence immediately.
