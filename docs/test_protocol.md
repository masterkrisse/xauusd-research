# Test Protocol

## Stage 1
Write the market behavior in plain English.

## Stage 2
Convert the behavior into exact machine-testable rules.

## Stage 3
Implement the simplest working baseline.

## Stage 4
Test across broad history with realistic costs.

## Stage 5
Check performance by year, session, long/short side, and regime.

## Stage 6
Check parameter stability.

## Stage 7
Test out-of-sample.

## Stage 8
Run walk-forward validation.

## Stage 9
Run hostile review for overfitting, leakage, execution fragility, and regime dependence.

## Stage 10
Add one improvement only, then retest.
