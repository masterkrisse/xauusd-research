# Result Presentation Format

Every strategy result must be presented in this fixed structure:

## 1. Executive Summary
- Strategy name
- Version
- Hypothesis family
- Test window
- Final verdict: Keep / Kill / Refine
- Confidence: Low / Medium / High
- One-paragraph conclusion

## 2. Core Metrics
- Net profit
- Profit factor
- Expectancy per trade
- Win rate
- Number of trades
- Max drawdown
- Recovery factor
- Return / drawdown

## 3. Robustness Metrics
- In-sample result
- Out-of-sample result
- OOS retention %
- Walk-forward stability
- Parameter stability
- Spread sensitivity
- Slippage sensitivity

## 4. Execution Realism
- Fill realism risk
- Fast-bar risk
- Broker/session dependency
- Time-window fragility

## 5. Regime Breakdown
- Compression
- Expansion
- Trend
- Reversal / Mean-reversion
- Shock / Event-risk

## 6. Directional Breakdown
- Long side metrics and verdict
- Short side metrics and verdict

## 7. Failure Analysis
- top overfitting risk
- top execution risk
- top regime risk
- most fragile parameter
- strongest argument against the strategy

## 8. Decision Block
- Keep / Kill / Refine
- exact reason
- one approved next change only
- rejected next changes
