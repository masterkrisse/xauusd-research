# Metrics Definition

## Core metrics
- Net profit
- Profit factor
- Expectancy per trade
- Maximum drawdown
- Number of trades
- Win rate
- Average trade
- Average R multiple
- Recovery factor
- Return / drawdown

## Robustness metrics
- Performance by year
- Performance by quarter
- Performance by session
- Long vs short performance
- Sensitivity to spread/slippage
- Sensitivity to parameter shifts
- Walk-forward stability
- Out-of-sample retention

## Decision standard
No strategy is accepted based on net profit alone.
A strategy must show acceptable robustness, execution realism, and parameter stability.
